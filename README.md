# Custom Marlin 3D Printer Firmware

This is reworked Marling 3D Printer firmware for a custom design 3D Printer.





##Parts list

#####Board - Ramps 1.4
#####Display - LCD Smart controller 12864
#####Stepper Motors - Stepper Motor 42 motor 17HS8401 1.8A
#####HotEnd - E3D 3mm
